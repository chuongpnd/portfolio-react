## Material UI Portfolio

A portfolio single page application using React and Material UI

#### Installing

#### To install dependency

```
npm install
```

#### To start the server

```
npm start
```

#### For Production Build

```
npm run build
```

Server will be available at http://127.0.0.1:3000 in your browser
